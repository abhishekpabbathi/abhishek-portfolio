# 🚀 Abhishek Portfolio

A modern, responsive developer portfolio built to showcase my software development journey, technical skills, and featured projects.

---

## ✨ Features

- Responsive design
- Dark & Light themes
- Smooth UI animations
- Featured projects showcase
- Skills section
- Contact form with EmailJS
- Mobile-friendly layout

---

### Frontend

- HTML5
- CSS3

### Libraries
- React Icons
- EmailJS

### Development Tools
- Git

- VS Code
